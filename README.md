### Description
This application is the back end of an e-commerce site. It uses a working Express.js API and configures it to use Sequelize to interact with a My SQL database.

### Installation
Use npm install to install the necessary dependencies, then use mysql -u root -p to open the mysql shell and run source db/schema.sql to create the database. Exit the mysql shell and run npm run seed to seed the database. Lastly, start the server by running npm start.

### Usage
You can use this app to look at different items for sale on an e-commerce site. You can view the products, categories and tags for each items and you can update, create, or delete any product, category, or tag as well.
